How to Trick Friends with "Hacks"



Do you need a new way to pull a good ol' fashioned prank your friends? In this tutorial, I will show you how to look like a hacker.

Part One: Setup

Go to Glitch.com and create a new account. Start a new project with the "hello-webpage template". This video will explain the next steps.

You can really name the file anything you want (name)/index.html. Now you should fill in the file with the content you want to make your friend think you hacked. For our example we just wrote "Hacks are turning on". Before we run the trick, make sure to change the project name of your Glitch project to somthing easy to remember.



Part Two: The Trick

For this step you will need a computer to open the terminal. To do this on Windows, click here. To do this on MacOS, click here. For Linux (Ubuntu), click here.
Once you have opened the terminal, run curl (project name).glitch.me/(file name). This will show the source of the website or in this case, what you typed into the Glitch editor.

And there you have it! Now you look like a "hacker". As always, use responsibly.
